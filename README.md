# RGBW-controller-with-timer-on-Arduino-Nano
Данное устройство предназначено для управления RGBW светодиодной лентой (12-24В, общий катод) по заранее заданной программе. Контроллер автоматически меняет цвета ленты (Красный, Зеленый, Синий, Белый) в заданной последовательности и с заданной длительностью, основываясь на показаниях встроенных часов реального времени (RTC).
