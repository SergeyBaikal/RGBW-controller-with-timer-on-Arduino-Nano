# RGBW controller with timer on Arduino Nano

# 1. ОПИСАНИЕ УСТРОЙСТВА

Данное устройство предназначено для управления RGBW светодиодной лентой (12-24В) по заранее заданной программе. Контроллер автоматически меняет цвета ленты (Красный, Зеленый, Синий, Белый) в заданной последовательности и с заданной длительностью, основываясь на показаниях встроенных часов реального времени (RTC).

**Основные возможности:**
*   Автоматическая смена 4-х цветов (RED, GREEN, BLUE, WHITE) и выключения (Темнота «DARK») по расписанию.
*   Возможность пропуска ненужных шагов (SKIP).
*   Раздельная регулировка яркости для каждого цвета (0-255).
*   Работа циклически или однократно.
*   Интуитивное управление с помощью энкодера.
*   LCD 1602 дисплей с настраиваемым отключением подсветки.
*   Сохранение всех настроек в энергонезависимую память (EEPROM).
*   Высокая настраиваемая частота ШИМ (до 31.2 kHz в режиме Phase-correct PWM и до 62.5 kHz в режиме Fast PWM) для исключения мерцания ленты.

8.  Аккумулятор 18650 (по необходимости, можно не устанавливать)
9.  Плата зарядки аккумулятора TP4056 (по необходимости, можно не устанавливать)
10. Понижающий модуль с 12/24 вольт до 5 вольт (LM2596)
11. Повышающий модуль CN6009 (если хотите сделать чтобы Arduino продолжало работать, когда отключат электроэнергию), (по необходимости, можно не устанавливать)

# 4. ПРОШИВКА КОНТРОЛЛЕРА

**Важно!** Отключайте внешнее питание при загрузке скетча (блок питания и аккумулятор, если подключены).

1.  Установите Arduino IDE на ваш компьютер.
2.  Установите необходимые библиотеки. Для этого откройте менеджер библиотек (Скетч -> Подключить библиотеку -> Управлять библиотеками...) и найдите/установите следующие библиотеки:

    *   `GyverTimers` (by AlexGyver)

3.  Подключите Arduino Nano к компьютеру через USB.
4.  В Arduino IDE выберите настройки:
    *   **Плата:** Инструменты -> Плата -> Arduino Nano
    *   **Процессор:** Инструменты -> Processor -> ATmega328P (Old Bootloader) (чаще всего)
    *   **Порт:** Инструменты -> Port -> COM... (нужный порт)
5.  Скопируйте код в окно Arduino IDE или откройте файл с расширением `.ino`. Примечание: файл ino должен лежать в одноименной папке, например RGBW.ino должен лежать в папке RGBW. 
6.  Нажмите кнопку **"Загрузить"** (стрелка вправо). Дождитесь окончания загрузки.
7.  Отключить от USB (!) если хотите сразу подключить к блоку питания, но можно не подключать блок питания (и аккумулятор), а потестировать работоспособность энкодера и управление через меню.

# 5. ПЕРВОЕ ВКЛЮЧЕНИЕ И НАСТРОЙКА

1.  Подайте питание на устройство.
2.  На экране появится заставка с версией прошивки, затем устройство перейдет в главное меню.
3.  **Важно:** если часы реального времени (RTC) не обнаружены, на экране появится ошибка **"RTC ERROR!"**. Проверьте подключение модуля и перезагрузите устройство (путём отключения питания Arduino).
4.  **Текущее время установится автоматически при компиляции, если нет, то:**
    *   Вращайте энкодер, чтобы выбрать пункт **«CLOCK»** в главном меню.
    *   Нажмите на кнопку энкодера, чтобы войти в подменю.
    *   Нажмите еще раз, чтобы начать редактирование часов. Мигающий символ **«H»** показывает, что редактируются часы. Вращайте энкодер для установки значения.
    *   Нажмите кнопку еще раз. Теперь мигает **«M»** — редактируются минуты. Вращайте энкодер.
    *   Нажмите кнопку для подтверждения и выхода из режима редактирования.
5.  Настройте цепочку цветов, их яркость и длительность (см. следующий раздел).

# 6. ИНТЕРФЕЙС И НАВИГАЦИЯ

Управление устройством осуществляется с помощью одного энкодера. Он имеет три основных действия:

*   **ВРАЩЕНИЕ:** Перемещение курсора в меню (вверх/вниз, влево/вправо) или изменение значения параметра.
*   **КОРОТКОЕ НАЖАТИЕ (КЛИК):** Вход в подменю, выбор пункта, начало/конец редактирования параметра.
*   **ДОЛГОЕ НАЖАТИЕ (УДЕРЖАНИЕ ~1 сек):** Выход из текущего меню на уровень выше или остановка активного режима RUN.

**Структура навигации:**
*   **Главное меню:** Список доступных разделов. Вращайте энкодер для выбора пункта, нажмите для входа.
*   **Подменю:** Список параметров внутри раздела. Вращайте энкодер для выбора параметра, нажмите для редактирования.
*   **Режим редактирования:** Значение параметра мигает. Вращайте энкодер для изменения значения. Нажмите для сохранения и выхода.

# 7. ДЕТАЛЬНОЕ ОПИСАНИЕ МЕНЮ

Главное меню состоит из 8 пунктов. Текущий пункт отмечен символом **«>»** слева. В нижней строке отображается следующий пункт для удобства навигации.

### 1. CHAIN (Цепочка цветов)
Здесь настраивается последовательность из 6 шагов.
*   **На экране:** `1#: R` (означает: шаг №1, цвет R).
*   **Редактирование:** Выберите нужный шаг (`1#`-`6#`) вращением, нажмите кнопку. Каждое нажатие меняет цвет по циклу:
    *   `R` (Красный)
    *   `G` (Зеленый)
    *   `B` (Синий)
    *   `W` (Белый)
    *   `D` (DARK, темнота/Выкл)
    *   `X` (SKIP, пропустить шаг)

### 2. BRIGHTNESS (Яркость)
Настройка яркости для каждого цвета индивидуально.
*   **На экране:** `R: 128` (означает: Красный канал, яркость 128).
*   **Редактирование:** Выберите канал (`R`, `G`, `B`, `W`) вращением, нажмите кнопку. Вращайте энкодер для изменения яркости от 0 до 255. Нажмите кнопку для сохранения. Во время редактирования выбранный цвет загорается для визуального контроля.

### 3. TIME (Длительность)
Установка времени работы для каждого шага цепочки.
*   **На экране:** `01:30 1#:R` (означает: 1 час 30 минут для шага №1 цвета R).
*   **Редактирование:** Выберите шаг (`1#`-`6#`) вращением, нажмите кнопку. Начните редактирование часов (мигает **«H»**), вращайте. Нажмите кнопку для перехода к минутам (мигает **«M»**), вращайте. Нажмите кнопку для сохранения.
*   **Примечание:** если для шага в меню «CHAIN» выбран цвет **«X»** (SKIP), его длительность автоматически устанавливается в **«00:00»** и не редактируется.

### 4. REPEAT (Повтор)
Определяет, будет ли цепочка цветов повторяться бесконечно.
*   **На экране:** **«YES»** или **«NO»**.
*   **Редактирование:** Нажмите кнопку для переключения значения.
    *   **«YES»:** после 6-го шага цикл начнется снова с 1-го.
    *   **«NO»:** после завершения 6-го шага работа остановится, лента погаснет.

### 5. CLOCK (Настройка часов)
Установка текущего времени на модуле DS3231.
*   **На экране:** **«SET CLOCK»** и текущее время (например, **«14:25»**).
*   **Редактирование:** Нажмите кнопку для входа в режим редактирования. Аналогично меню «TIME»: сначала редактируются часы, затем минуты.
*   Если RTC не обнаружен, на экране будет **«RTC ERROR»**.

### 6. TEST (Тест)
Быстрая проверка работоспособности ленты и подключения.
*   **Запуск:** Войдите в меню и нажмите кнопку. Устройство по очереди включит на 1 секунду каждый цвет (Красный, Зеленый, Синий, Белый), а затем погаснет.

### 7. DISPLAY (Дисплей)
Настройка таймера подсветки дисплея для экономии энергии.
*   **На экране:** текущий режим.
*   **Редактирование:** Нажмите кнопку для циклического переключения режимов:
    *   **«OFF»:** Подсветка всегда выключена.
    *   **«15MIN»:** Подсветка гаснет через 15 минут бездействия (вращения/нажатия энкодера).
    *   **«1HOUR»:** Подсветка гаснет через 1 час бездействия.
    *   **«ALWAYS»:** Подсветка всегда включена.

### 8. RUN (Запуск)
Основной режим автоматической работы по расписанию.
*   **Запуск:** Войдите в меню. Если RTC исправен, режим запустится автоматически.
*   **Экран RUN:**
    *   Верхняя строка: **«RUN 14:35»** (текущее время).
    *   Нижняя строка: **«01:23 3#:B»** (оставшееся время до смены цвета: 1 час 23 минуты, текущий шаг №3, цвет Синий).
*   **Во время работы:**
    *   **Клик:** показывает на 5 секунд всю цепочку цветов (например, **`R-G-B-W-D-X`**).
    *   **Удержание:** останавливает режим RUN и возвращает в главное меню.

Автоматика сама будет переключать цвета в заданное время, основываясь на времени старта шага и его длительности.

# 8. РЕЖИМ РАБОТЫ ПО РАСПИСАНИЮ (RUN)

Это сердце программы. Алгоритм работы:

1.  При входе в меню **«RUN»** фиксируется текущее время (минуты) как время старта первого шага (Step 1).
2.  Контроллер включает цвет, заданный для Step 1.
3.  Каждую минуту программа проверяет, сколько времени прошло с момента старта шага.
4.  Как только пройденное время становится равным или превышает заданную для этого шага длительность (из меню «TIME»), происходит автоматический переход к следующему шагу.
5.  Шаги, помеченные как **«X»** (SKIP), автоматически пропускаются (на них не тратится время).
6.  Если после завершения последнего шага параметр «REPEAT» установлен в **«YES»**, цикл начинается заново с шага 1. Если **«NO»** — режим RUN останавливается, лента гаснет.

# 9. СБРОС НАСТРОЕК

Если вы хотите вернуть все настройки (цепочку, яркость, время и т.д.) к предустановленным значениям:

1.  Убедитесь, что устройство включено.
2.  Нажмите и удерживайте тактовую кнопку, подключенную к пину D7 (Reset).
3.  На экране появится сообщение **«RESET MEMORY»**.
4.  Отпустите кнопку. Настройки будут сброшены, и устройство перезагрузится.

*Примечание: функцию сброса можно не использовать, я сделал её для себя, чтобы быстро возвращаться к первоначальным параметрам.*

**Предустановленные настройки:**
*   **Цепочка:** R, G, B, W, D, X
*   **Яркость всех цветов:** 128
*   **Длительность всех шагов:** 120 минут (v.1.8.4), 2 минуты (v.1.8.5)
*   **Repeat:** YES
*   **Подсветка:** 15MIN
